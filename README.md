# next-js-coffee-website

Bu proje, modern web geliştirme pratiklerini uygulamak amacıyla JavaScript temelli Next.js kullanılarak geliştirilmiş bir web sitesi çalışmasıdır. Proje; component yapısı, sayfa yönlendirmeleri (routing), global stiller ve tüm siteyi kapsayan responsive tasarım gibi temel Next.js/React prensipleri esas alınarak hazırlanmıştır.

---

## Kullanılan Teknolojiler

- Next.js
- React
- JavaScript (ES6+)
- CSS
- Responsive Design (tüm sayfa yapısı)
- Vercel (Deployment)

---

## Özellikler

- Component tabanlı mimari
- Next.js pages ve routing yapısı
- Global CSS dosya yönetimi
- Tüm sayfa boyunca mobil uyumlu (responsive) tasarım
- Responsive navbar
- Temiz ve sade UI
- Optimize edilmiş dosya ve klasör yapısı
